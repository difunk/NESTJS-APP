# NESTJS-APP

![CI](https://github.com/difunk/NESTJS-APP/actions/workflows/ci.yml/badge.svg)

This project contains a NestJS backend and a React frontend (quote-react-app).

## Project Structure

- `src/`: NestJS backend
- `quote-react-app/`: React frontend

## Getting Started

### Backend

```bash
npm install
npm run start:dev
```

### Frontend

```bash
cd quote-react-app
npm install
npm run dev
```
